ISS Location

This page tracks and displays the real-time location of the International Space Station (ISS), showcasing its current latitude and longitude. Then, mapping it live on a map.

Coordinates

This project uses web development technologies, including HTML, CSS, JavaScript, and Bootstrap, along with a RESTful API (Where the ISS At API) to retrieve live data.

This project demonstrates how to integrate asynchronous JavaScript functions (async/await) with an external API to fetch and display dynamic data in a user-friendly format.

API used: https://api.wheretheiss.at/v1/satellites/25544
