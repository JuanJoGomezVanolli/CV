NASA (NEO) API

  • This project shows the top 5 biggest near earth objects that passed on a specific day.
  • The page takes the input date with the format YYYY-MM-DD and makes the request on the API provided by NASA, with a fetch() method.
  • API Link: https://api.nasa.gov/neo/rest/v1/feed?start_date=${input}&end_date=${input}&api_key=YOUR_API_KEY
  • Then the page takes the JSON response and using an array and sorting algorith prints the top 5 objects by their Magnitude.
