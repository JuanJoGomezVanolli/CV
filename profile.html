Misc JavaScrypt concepts...

Sets

Set objects are collections of values. A value in the set may only occur once; it is unique in the set's collection.

can iterate through the elements of a set in insertion order.

The insertion order corresponds to the order in which each element was inserted into the set by the add() method successfully (that is, there wasn't an identical element already in the set when add() was called).

Maps

The Map object holds key-value pairs and remembers the original insertion order of the keys. Any value (both objects and primitive values) may be used as either a key or a value.
